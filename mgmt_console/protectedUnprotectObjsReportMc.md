# Protected or unprotected VMs
{: #report_protected_unprotected_objects_mc}

The Protected or unprotected VMs report provides lists of VMs protected and not protected by {{site.data.keyword.baas_full_notm}}.

- [Filter Report Data](#report_protected_unprotected_vm_filter_mc)
- [Summary](#report_protected_unprotected_vm_summary_mc)





## Filter Report Data
{: #report_protected_unprotected_vm_filter_mc}

The report supports filters to narrow down the data for Unprotected VMs or Protected VMs that you want to see.




## Summary
{: #report_protected_unprotected_vm_summary_mc}

The report includes the following details:

- Sources Registered: Total number of sources.
- Total VMs: Total number of virtual machines.
- Protected VMs: Total number of protected virtual machines.
- Unprotected VMs: Total number of unprotected virtual machines.
- Unprotected Size: The total amount of unprotected data, in bytes.
