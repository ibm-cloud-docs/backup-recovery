# Protection runs summary
{: #report_protection_runs_mc}

The Protection Runs report provides a summary and list of all backup activities per object per run. You can view the summary and success rate of protection runs. You can also view the snapshot status of the protection run.



- [Filter Report Data](#report_protection_run_filter_mc)
- [Report Data](#report_data_protection_run_mc)


## Filter Report Data
{: #report_protection_run_filter_mc}

The report supports a date range filter to pare down the data that you want to view in the report:


| Object Name | Description |
|-------------------------|--------------------------------------------------------------|
| Date Range	    | The start and end dates. |


## Report Data
{: #report_data_protection_run_mc}

The following table describes the data that is displayed in the Data table.


| Column Name | Description |
|-------------------------|--------------------------------------------------------------|
| Protection Group name |	The protection group that is associated with the protection run for the corresponding object. |
| Start Time    | The date and time when the protection run started. |
| Duration      |	The time taken to complete the protection run. |
| Bytes Read |	The total amount of data read, in bytes. |
| Status |	The status of the snapshot. |
